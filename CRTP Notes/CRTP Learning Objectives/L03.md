* Display name of the GPO applied on StudentMachines OU

```
Students
```

---

First let's list all the OU present in the domain using command :

```
Get-DomainOU | select name
```

```
name
----
Domain Controllers
StudentMachines
Applocked
Servers
DevOps
```

Now from here we can enumerate GPO applied on StudentMachines OU :

```
(Get-DomainOU -Identity StudentMachines).gplink
```

Then use that gplink for further enumerating
```
Get-DomainGPO -Identity "{7478F170-6A0C-490C-B355-9E4618BC785D}"
```

```
flags : 0
displayname : Students

..

..
```
