* Script used for hunting for admin privileges using PowerShell Remoting

```
Find-PSRemotingLocalAdminAccess.ps1
```

--- 

Solution :

`Find-PSRemotingLocalAdminAccess.ps1` Tool can be used for hunting admin priv 

```
Find-PSRemotingLocalAdminAccess
```

Output:
```
dcorp-adminsrv
```

Which says we have admin access on `dcorp-adminsrv`
We can connect to `dcorp-adminsrv` using `winrs` as the student user:

```
winrs -r:dcorp-adminsrv cmd
```

check
```
$env:computername
```

```
DCORP-ADMINSRV 
```
